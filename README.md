# common-utils-js


## Installation

Using npm:
```shell
$ npm i --save common-utils-js
```

In Node.js:
```js
// Load the full build.
const __ = require('common-utils-js');
```

## Why common-utils-js?

Commonly used functions in javascript. common-utils-js makes JavaScript more easier.

## Examples
```js
__.isObjectEmpty({});
true;
```
etc..
